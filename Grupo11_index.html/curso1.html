Curso de Desarrollador Web

Descripción del Curso

Bienvenido al curso de Desarrollador Web del Grupo 11. Este curso está diseñado para proporcionarte las habilidades necesarias para convertirte en un desarrollador web profesional. A lo largo del curso, aprenderás sobre HTML, CSS, JavaScript y otros marcos y tecnologías importantes.

Detalles del Curso

  • Duración: 12 semanas
  • Modalidad: Online
  • Tipo: Curso grabado con sesiones en vivo semanales para resolver dudas

Modalidad del Curso

El curso se imparte de manera online, combinando materiales grabados que puedes seguir a tu propio ritmo con sesiones en vivo semanales donde podrás interactuar con los instructores y resolver cualquier duda que tengas.
